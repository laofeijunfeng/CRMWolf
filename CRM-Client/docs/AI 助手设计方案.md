# AI对话页面（新对话空状态）布局与设计方案
本方案基于截图的AI对话初始页拆解，覆盖**布局架构、分模块设计、UI视觉规范、交互体验设计**四大核心维度，可直接用于AI助手产品的前端开发与UI还原。

## 一、整体布局架构
页面采用**「上下分层、居中对称、核心聚焦」**的极简单栏布局，是AI对话产品典型的「新对话/空会话」页面设计，无展开的侧边栏，整体符合桌面端对话产品的用户操作习惯，核心结构分为3个固定模块，从上到下依次为：
1.  顶部通栏标题区（静态页头）
2.  居中核心主内容区（空状态视觉核心）
3.  底部固定输入操作区（核心交互入口，不随页面滚动）

整体布局核心原则：
- 视觉重心集中在页面垂直中上部，符合用户从上到下的浏览习惯
- 核心操作区固定在视口底部，用户无需跳转视线即可完成输入
- 全页面采用弹性响应式设计，适配不同宽度的桌面端屏幕
- 极致留白设计，无冗余元素，所有设计都服务于「引导用户发起对话」的核心目标

## 二、分模块详细布局与设计
### 模块1：顶部通栏标题区
- **位置**：页面最顶部，通栏全宽布局，与页面主体共用纯白背景，无分割线、无阴影
- **布局结构**：左右+居中的三分结构，左对齐功能按钮，居中标题文案
  1.  左侧：极简功能图标（返回/收起侧边栏按钮），左对齐，距离页面左边缘有固定安全边距
  2.  居中：双行文案，主标题「新对话」+ 辅助说明，完全水平居中
- **设计细节**：
  - 文字层级：主标题使用中等字号、常规字重，深灰色主色；辅助说明使用小字号、浅灰色，弱化存在感，不抢视觉重心
  - 留白：顶部预留充足的上间距，与下方主内容区拉开明显的视觉距离，保证页面呼吸感
  - 视觉克制：无背景色、无边框、无多余装饰，完全融入页面整体

### 模块2：居中核心主内容区（空状态核心）
- **位置**：页面垂直方向黄金分割线偏上位置，完全水平居中，是页面的视觉锚点
- **布局结构**：上下双层垂直布局，上层为欢迎标题，下层为推荐快捷prompt按钮组，两层之间预留充足的垂直间距
  1.  欢迎语标题层
      - 内容：「有什么我能帮你的吗？」，单句文案，完全居中
      - 设计：大号字体+加粗字重，使用最深的主文字色，作为空状态的视觉核心，一眼抓住用户注意力
  2.  推荐快捷prompt按钮层
      - 布局：采用Flex弹性流式布局，自动换行，整体水平居中，按钮之间保留固定的水平/垂直间距，截图内为3行3列的均匀排布
      - 按钮样式：中等圆角矩形，弱填充（极浅灰色背景），无边框，内边距充足，保证点击热区舒适；按钮内文字为常规字重，深灰色，口语化短句
      - 内容特性：按钮文案均为用户高频提问场景，降低用户输入门槛，实现一键发起对话
- **设计细节**：
  - 间距系统：标题与按钮组、按钮行与行之间，均采用统一基准的间距，无拥挤感，保证页面通透
  - 视觉层级严格区分：欢迎标题 > 按钮文字，层级清晰，无视觉干扰
  - 响应式适配：屏幕宽度变化时，按钮自动换行，始终保持整体居中，不出现横向滚动

### 模块3：底部固定输入操作区
- **位置**：视口最底部，通栏固定定位，不随页面滚动，始终保持在用户可视范围内，是产品的核心操作入口
- **布局结构**：大圆角整体输入容器，采用左-中-右横向布局，输入区域占满容器剩余宽度，左右功能按钮对称排布
  1.  左侧附加功能区
      - 核心入口：圆形功能按钮，点击弹出垂直下拉气泡菜单，整合所有附加功能
      - 下拉菜单：白色背景+轻微悬浮阴影，左对齐列表，每个选项搭配图标+文字，包含「上传文件或图片」等核心功能，菜单贴合按钮上方弹出，不遮挡输入框
  2.  中间输入核心区
      - 容器：无边框输入框，与外层整体容器融为一体，占满中间全部剩余宽度
      - 内部设计：输入框左侧预留功能标签位，右侧为文本输入区域，支持多行输入自适应
      - 交互态：输入容器默认带淡色描边，聚焦时高亮描边，提升操作反馈
  3.  右侧快捷功能区
      - 核心入口：麦克风图标按钮，语音输入功能，与左侧按钮视觉对称，保证整体平衡
- **设计细节**：
  - 容器设计：大圆角矩形，极浅灰色背景，左右预留安全边距，不贴满屏幕边缘，设置最大宽度限制，超宽屏幕下始终居中
  - 功能聚合：非核心的上传功能全部折叠在下拉菜单中，既保证功能完整性，又不破坏界面的极简整洁
  - 操作适配：输入框支持多行文本，高度随输入内容自适应，保证长文本输入的可读性

## 三、UI视觉设计规范
### 1. 色彩系统
整体采用**中性色为主，低饱和辅助色点缀**的配色方案，符合AI产品专业、简约、科技的调性，无多余色彩干扰。
| 色彩层级 | 应用场景 | 色彩参考 |
| :--- | :--- | :--- |
| 主背景色 | 页面整体背景 | 纯白 #FFFFFF |
| 一级文字色 | 欢迎标题、核心文案 | 深黑/极深灰 #121212 |
| 二级文字色 | 按钮文字、页头标题、输入框文字 | 深灰 #333333 |
| 三级文字色 | 辅助说明、占位符、弱化文案 | 中灰 #666666 / #999999 |
| 组件背景色 | 按钮、输入框默认背景 | 极浅灰 #F5F5F5 / #F8F8F8 |
| 交互辅助色 | 输入框聚焦描边、hover高亮 | 低饱和淡紫色 |
| 悬浮阴影 | 下拉菜单 | 极低不透明度的黑色柔化阴影 |

### 2. 字体与排版规范
- 字体选型：优先选用跨平台兼容的无衬线字体（桌面端优先Inter、思源黑体、苹方、微软雅黑等系统默认无衬线字体），保证全场景可读性
- 字号与字重层级（桌面端）：
  - 欢迎大标题：24-32px，字重700（加粗）
  - 页头主标题：16-18px，字重500（中等）
  - 按钮文字、输入框文字：14-15px，字重400（常规）
  - 辅助说明文字：12-13px，字重400（常规）
- 对齐规则：核心内容全部水平居中对齐，仅下拉菜单、输入框内文本左对齐，整体视觉高度统一

### 3. 圆角与间距系统
- 圆角规范：全页面采用统一的圆角值，按钮、输入框、下拉菜单圆角统一为8-12px，兼顾亲和力与专业感，无极端全圆角或直角设计
- 间距系统：采用8px为基准的栅格间距，保证全页面间距统一
  - 按钮内边距：12px 16px
  - 按钮之间间距：水平12px，垂直16px
  - 模块之间留白：32px / 48px / 64px
  - 页面安全边距：左右最小16px，超宽屏幕下设置最大内容宽度，居中限制

### 4. 质感与动效规范
- 质感设计：极致扁平化风格，无渐变、无复杂纹理、无厚重阴影，仅下拉菜单使用轻微悬浮阴影，输入框使用细微描边，整体轻量化、干净通透
- 动效设计：极简微交互，无复杂动画，仅保留核心操作反馈
  - 按钮hover：背景色平滑过渡（150ms缓动）
  - 下拉菜单：弹出/收起的淡入淡出动画（200ms缓动）
  - 输入框聚焦：描边高亮的平滑过渡（150ms缓动）

## 四、交互与用户体验设计
1.  **空状态引导设计**：新对话无历史消息时，通过欢迎语+高频推荐prompt，降低用户使用门槛，解决用户「不知道问什么」的痛点，点击推荐按钮可一键填充并发起对话
2.  **核心操作优先级**：底部固定输入框始终可见，用户随时可以输入，视线无需来回跳转，符合对话产品的操作直觉
3.  **功能折叠设计**：非核心的上传/附加功能全部折叠在下拉菜单中，主界面保持极简，同时不损失功能完整性，兼顾新手用户与进阶用户的需求
4.  **状态切换逻辑**：用户发起对话后，主内容区自动切换为对话消息流，推荐prompt消失，输入框保持在底部，符合对话产品的常规使用逻辑
5.  **响应式适配**：弹性布局适配全尺寸桌面端屏幕，窄屏时推荐按钮自动换行，输入框自适应宽度，无横向滚动，保证全场景可用性

## 五、核心设计亮点
1.  **极致极简主义**：无冗余视觉元素，无广告、无多余功能按钮，所有设计都服务于「发起对话」的核心目标，用户注意力无分散
2.  **对称平衡美学**：整体布局左右对称，标题居中、主内容居中、输入框功能按钮对称，视觉平衡感极强，观感舒适
3.  **严格的视觉层级**：全页面仅3个核心视觉层级，无多余层级干扰，用户浏览无负担，一眼就能找到核心内容与操作入口
4.  **克制的功能设计**：核心功能优先，附加功能隐藏，既保证了界面的简洁，又满足了进阶用户的使用需求，兼顾易用性与功能性
