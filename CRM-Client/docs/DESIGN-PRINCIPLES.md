# CRMWolf 设计原则

⚠️ **品牌色唯一来源**：`src/styles/variables.scss`

**所有颜色必须引用 Sass 变量，禁止硬编码。颜色定义不在本文档中。**

---

## 一、设计原则

### 1.0 Signature 元素（独特视觉标识）

根据 frontend-design skill 要求：每个产品应有"让人记住"的独特元素。

| Signature | 实现 | 位置 |
|-----------|------|------|
| **AI 对话区** | IBM Plex Mono 字体 + 微蓝背景 + 智能边线 | `ChatBubble.vue` |
| **智能边界线** | 流动感渐变线条 | `AIAssistant.vue` 输入区顶部 |
| **不对称圆角** | 用户：左下角直角；AI：右下角直角 | `ChatBubble.vue` |

### 1.1 核心原则

| 原则 | 说明 |
|------|------|
| **简约至上** | 去除多余装饰，保留核心信息 |
| **层次分明** | 浅灰背景 + 白色卡片建立清晰层级 |
| **轻量高效** | 柔和阴影、细腻圆角、流畅动效 |
| **一致统一** | 全系统使用同一套设计 token（variables.scss） |
| **信息密度** | 在保证可读性的前提下，提高信息密度 |
| **克制 Display font** | IBM Plex Sans 仅用于页面标题（≤5%） |

### 1.2 命名空间

所有 CSS 类名使用 `wolf-` 前缀：

```css
.wolf-card      /* 卡片 */
.wolf-tag       /* 标签 */
.wolf-btn       /* 按钮 */
.wolf-table     /* 表格 */
.wolf-layout    /* 布局 */
.wolf-ai-message /* AI 消息（← Signature） */
```

## 二、字体系统

### 2.1 字号层级（已统一）

| 用途 | 字号 | 字重 | 字体 |
|------|------|------|------|
| 页面大标题 | 24px | 600 | IBM Plex Sans |
| 页面标题 | 20px | 600 | IBM Plex Sans |
| 卡片标题 | 16px | 600 | 系统字体栈 |
| 正文 | 14px | 400 | 系统字体栈 |
| AI 消息 | 14px | 400 | **IBM Plex Mono**（← Signature） |
| 小字 | 13px | 400 | 系统字体栈 |
| 标签/辅助 | 12px | 400 | 系统字体栈 |

### 2.2 字重规范

| 字重 | 用途 |
|------|------|
| 400 | 正文 |
| 500 | 强调 |
| 600 | 标题 |

### 2.3 行高

| 场景 | 行高 |
|------|------|
| 标题 | 1.25 |
| 正文 | 1.5 |

## 三、阴影系统

| 用途 | 阴影值 |
|------|--------|
| 静态卡片 | `0 1px 2px rgba(0,0,0,0.04)` |
| 悬停状态 | `0 4px 12px rgba(0,0,0,0.08)` |
| 下拉菜单 | `0 8px 24px rgba(0,0,0,0.12)` |
| 弹窗/抽屉 | `0 16px 48px rgba(0,0,0,0.16)` |

## 四、设计红线

| 红线 | 说明 |
|------|------|
| 禁止硬编码颜色 | 必须引用 variables.scss 变量 |
| 禁止纯色填充标签 | 必须用浅底色+同色系文字 |
| 禁用 700+ 字重 | 最大使用 semibold (600) |
| 禁止高饱和色 | 使用 variables.scss 低饱和色系 |
| 禁止多层阴影 | 卡片仅用一级阴影 |
| 禁止魔数间距 | 向 4px 基准网格对齐 |
| **禁止卡片用 Display font** | 卡片标题使用系统字体栈 |

---

## 五、间距命名规范

**仅使用语义化命名**，废弃数字命名：

| 变量 | 值 | 用途 |
|------|-----|------|
| `$wolf-space-xs` | 4px | 元素内间距（图标与文字） |
| `$wolf-space-sm` | 8px | 关联元素间距 |
| `$wolf-space-md` | 16px | 模块内间距（卡片内边距） |
| `$wolf-space-lg` | 24px | 模块间间距、页面安全边距 |

❌ **废弃**：`$wolf-space-1/2/3/4/5/6/8/10/12`（数字命名）
