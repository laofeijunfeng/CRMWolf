## 七、状态码与标签对应

### 线索状态

| 状态值 | 文字 | 标签色 |
|--------|------|--------|
| 0 | 新建 | 信息色 |
| 1 | 跟进中 | 警告色 |
| 2 | 已转化 | 成功色 |
| 3 | 无效 | 错误色 |

### 商机状态

| 状态值 | 文字 | 标签色 |
|--------|------|--------|
| 0 | 跟进中 | 警告色 |
| 1 | 已赢单 | 成功色 |
| 2 | 已输单 | 错误色 |

### 合同状态

| 状态值 | 文字 | 标签色 |
|--------|------|--------|
| DRAFT | 草稿 | 默认色 |
| PENDING | 待审批 | 警告色 |
| APPROVED | 已通过 | 成功色 |
| REJECTED | 已拒绝 | 错误色 |

## 八、设计红线汇总

| 红线 | 说明 |
|------|------|
| 禁止 any 类型 | 使用 Zod 校验 |
| 禁止纯色标签 | 浅底色+同色系文字 |
| 禁止 700+ 字重 | 最大 600 |
| 禁止竖分割线 | 仅行分割 |
| 禁止斑马纹 | hover 高亮 |
| 禁止高对比背景 | 柔和色差 |
