# 数据库设计

## 1. 数据库概述

- **数据库类型**：MySQL 8.0
- **字符集**：utf8mb4
- **引擎**：InnoDB

## 2. 数据表清单

| 表名 | 说明 |
|------|------|
| users | 用户表 |
| roles | 角色表 |
| permissions | 权限表 |
| user_roles | 用户角色关联表 |
| role_permissions | 角色权限关联表 |
| crm_leads | 线索表 |
| crm_lead_follow_ups | 线索跟进记录表 |
| crm_customers | 客户表 |
| crm_contacts | 联系人表 |
| crm_customer_follow_ups | 客户跟进记录表 |
| crm_opportunities | 商机表 |
| crm_opportunity_stages | 商机阶段表 |
| crm_contracts | 合同表 |
| crm_contract_payment_plans | 回款计划表 |
| crm_payment_records | 回款记录表 |
| crm_invoice_titles | 发票抬头表 |
| crm_invoice_applications | 发票申请表 |
| crm_approvals | 审批记录表 |
| crm_approval_flows | 审批流程表 |
| crm_operation_logs | 操作日志表 |

## 3. 核心数据表设计

### 3.1 用户表 (users)

| 字段名 | 类型 | 约束 | 说明 |
|--------|------|------|------|
| id | INT | PK, AUTO_INCREMENT | 主键 |
| feishu_open_id | VARCHAR(100) | UNIQUE, NOT NULL | 飞书open_id |
| feishu_union_id | VARCHAR(100) | INDEX | 飞书union_id |
| feishu_user_id | VARCHAR(100) | INDEX | 飞书user_id |
| name | VARCHAR(100) | NOT NULL | 用户姓名 |
| en_name | VARCHAR(100) | | 用户英文名 |
| email | VARCHAR(255) | INDEX | 用户邮箱 |
| mobile | VARCHAR(20) | | 用户手机号 |
| avatar_url | VARCHAR(500) | | 用户头像URL |
| employee_no | VARCHAR(50) | | 用户工号 |
| tenant_key | VARCHAR(100) | | 企业标识 |
| region | VARCHAR(50) | | 所属地区 |
| status | ENUM | DEFAULT 'active' | 用户状态 |
| created_at | DATETIME | DEFAULT NOW | 创建时间 |
| updated_at | DATETIME | ON UPDATE NOW | 更新时间 |
