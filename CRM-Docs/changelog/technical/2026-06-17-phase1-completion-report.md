## 🚀 批量优化脚本（已准备）

### 脚本 1：批量添加 import

```bash
#!/bin/bash
# add-imports-batch.sh

FILES=(
  "CRM-Client/src/views/Opportunities.vue"
  "CRM-Client/src/views/Contracts.vue"
  "CRM-Client/src/views/CustomerDetail.vue"
  # Phase 2-3 所有文件
)

for file in "${FILES[@]}"; do
  if ! grep -q "import { showError, showSuccess }" "$file"; then
    sed -i '' '/import { ElMessage } from '\''element-plus'\''/a\
import { showError, showSuccess } from '\''@/utils/errorMessages'\''
' "$file"
    echo "✅ Added imports: $file"
  fi
done
```

### 脚本 2：批量替换示例

```bash
#!/bin/bash
# replace-messages-batch.sh

# Contracts.vue
```
