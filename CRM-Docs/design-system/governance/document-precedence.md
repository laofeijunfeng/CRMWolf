# 文档优先级

- **用途：**确定规则的唯一事实来源与冲突处理。
- **适用范围：**设计系统全部主题文档。
- **权威性：**本文件定义文档层级；具体规则由各主题文件拥有。
- **相关规范：**[写作规范](authoring.md) · [根入口](../README.md)

## 优先级

页面特有规则 > 交互模式 > 组件规则 > 基础规范。发生冲突时采用更高层级的页面特有规则，并修正同级重复或冲突内容。

## 单一事实来源

每条规则只由一个主题文件权威定义；其他文档使用相对链接引用，不复制数值、状态表或实现规则。同级文件不得存在冲突规则。

## 目标状态

本库描述目标状态。尚未落地的迁移内容必须明确标为目标状态，不得表述为当前实现。设计系统目录是唯一权威来源；发现的非权威旧总则应删除或归档，且不得被索引或引用。
