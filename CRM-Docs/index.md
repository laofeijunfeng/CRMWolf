## 系统概览

CRMWolf 是一套面向企业的客户关系管理系统，覆盖线索 → 客户 → 商机 → 合同 → 回款 → 发票完整销售链路。

### 核心模块

| 模块 | 核心功能 |
|------|----------|
| 线索管理 | 线索创建、跟进、转化、公海池 |
| 客户管理 | 客户档案、联系人、公海池、AI 智能档案 |
| 商机管理 | 商机创建、阶段推进、赢单/输单 |
| 合同管理 | 合同创建、审批流程、状态管理 |
| 回款管理 | 回款登记、分期管理、关联合同 |
| 发票管理 | 发票申请、审批、开具 |
| AI 功能 | AI 助手、意图识别、智能操作 |
