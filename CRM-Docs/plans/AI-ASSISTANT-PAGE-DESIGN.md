## 一、Ground it in the subject

### 1.1 产品定位

| 元素 | 定义 |
|------|------|
| **产品** | CRMWolf AI 助手 - 销售团队的智能操作助手 |
| **受众** | 销售团队（日常高频使用，追求效率） |
| **页面任务** | 一个任务：**快速执行业务操作** |

### 1.2 设计主题来源

从 CRM 系统的世界提取独特性：

| 来源 | 设计灵感 |
|------|----------|
| **业务卡片** | 侧边栏的对话列表借鉴客户卡片的紧凑排列 |
| **状态标签** | 预览卡片的操作类型标签借鉴系统已有的状态标签设计 |
| **输入框** | 对话输入借鉴系统搜索框的简洁风格 |
| **数据密度** | CRM 列表页的信息密度 → 对话历史也需要紧凑但可读 |

## 二、Design Principles

### 2.1 避免 AI 模板化设计

**拒绝三种 AI 默认模板**：

| 模板 | 特征 | 拒绝理由 |
|------|------|----------|
| (1) 奶油色背景 + 衬线字体 | #F4F1EA + 高对比衬线 + 赤陶强调色 | 系统已有暖调米白 #F9F8F5，但不配衬线字体 |
| (2) 黑色背景 + 酸性绿 | 近黑 + 单一亮色强调 | CRM 是商务系统，不适合暗色主题 |
| (3) 报纸式布局 + 零圆角 | hairline rules + dense columns | 系统已有圆角体系 ($wolf-radius-lg) |

### 2.2 签名元素

**唯一独特元素**：**左侧对话列表的"实体标签"**

```
历史对话项 = 对话标题 + 实体类型标签

┌────────────────┐
│ 创建客户张三   │ ← 对话标题（自动提取）
│ [客户]         │ ← 实体类型标签（同系统状态标签）
├────────────────┤
│ 商机赢单50万   │
│ [商机]         │
└────────────────┘
```

**独特性理由**：ChatGPT 的历史列表仅显示时间，无业务上下文。CRMWolf 的历史对话绑定业务实体，让销售一眼看到"这是关于哪个客户/商机的操作"。

## 三、Token System

### 3.1 色彩 Palette（4-6 named hex）

```
主背景：      #F9F8F5 (wolf-bg-page)     - 页面画布，暖调米白
卡片背景：    #FFFFFF (wolf-bg-card)      - 对话气泡、预览卡片
侧边栏背景：  #F2F0EB (wolf-bg-sidebar)  - 历史对话列表
主文字：      #1C1C1C (wolf-text-primary) - 标题、对话内容
辅助文字：    #3A3A3A (wolf-text-secondary) - 时间、参数标签
强调色：      #4A6FA5 (wolf-primary)      - 用户气泡、发送按钮（低饱和蓝）
成功色：      #2B633C + #EDF7EF           - 状态卡片成功
警告色：      #7A4F1E + #FFF6E8           - 状态卡片警告
```

### 3.2 Typography

```
页面标题：    16px / 600  (wolf-font-size-title)
对话内容：    14px / 400  (wolf-font-size-body)
辅助信息：    13px / 400  (wolf-font-size-auxiliary)
时间/标签：   12px / 400  (wolf-font-size-caption)
```

**字体选择**：使用系统已有字体 `$wolf-font-family`（系统字体栈），不引入新字体。

### 3.3 Layout

```
┌──────────────────────────────────────────────────────────────┐
│  Header: 100%                                                │
│  AI 助手                               [新对话]              │
├─────────────┬────────────────────────────────────────────────┤
│  侧边栏      │                                                │
│  280px      │  对话区                                        │
│  fixed      │  flex: 1                                       │
│             │                                                │
│  历史对话    │  max-width: 800px                              │
│  按日期分组  │  居中                                          │
│             │                                                │
│             │  欢迎界面 → 对话气泡 → 内嵌卡片 → 输入框        │
├─────────────┴────────────────────────────────────────────────┤
│  输入区: max-width 800px, 居中                                │
└──────────────────────────────────────────────────────────────┘
```

### 3.4 Signature

**签名元素**：历史对话项的实体标签

```
┌────────────────────┐
│ 创建客户张三        │ ← 标题
│ [客户] 14:30       │ ← 实体标签 + 时间
└────────────────────┘
```

## 五、响应式布局

### 5.1 大屏（≥1200px）

```
侧边栏：280px fixed 展开
对话区：flex: 1
输入区：max-width 800px
```

### 5.2 中屏（768px-1199px）

```
侧边栏：可折叠（默认展开）
折叠按钮：Header 新增 [≡] 按钮
对话区：flex: 1
输入区：max-width 100%, padding 16px
```

### 5.3 小屏（<768px）

```
侧边栏：默认隐藏
折叠按钮：Header 必需 [≡] 按钮
对话区：100% 全宽
输入区：100% 全宽, padding 12px
快捷操作：2x2 网格排列（而非横向）
```

### 6.1 模板化检查

| 元素 | 是否模板化 | 改进 |
|------|------------|------|
