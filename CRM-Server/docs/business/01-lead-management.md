# 线索管理流程

## 1. 线索生命周期

```
新增线索 → 跟进中 → 已转化 → 客户
              ↓
           已输单 → 公海池（可重新领取）
```

## 2. 线索状态说明

| 状态 | 值 | 说明 |
|------|-----|------|
| 新线索 | NEW | 刚创建的线索，尚未开始跟进 |
| 跟进中 | CONTACTING | 销售人员正在跟进的线索 |
| 已转化 | CONVERTED | 线索已转化为客户 |
| 已输单 | LOST | 确认无法成交，退回公海池 |

## 3. 线索来源

系统支持多种线索来源的记录和统计：

- 线上注册
- 市场活动
- 客户推荐
- 电话营销
- 网站咨询
- 展会
- 其他

## 4. 核心功能

### 4.1 线索创建

**创建方式：**
- 手动创建：销售人员在系统中录入线索信息
- 批量导入：通过Excel批量导入线索

**必填字段：**
- 线索名称
- 联系人姓名
- 联系电话
- 线索来源

### 4.2 线索分配

**分配规则：**
1. **自动分配**：系统根据规则自动分配给销售人员
   - 按地区平均分配
   - 按销售人员的负载均衡
   
2. **手动分配**：销售总监手动分配线索给指定销售人员

**分配限制：**
- 每个销售人员同时最多拥有50个活跃线索
- 公海池中的线索可以被任何销售人员领取

### 4.3 线索跟进

**跟进记录：**
- 销售人员必须定期添加跟进记录
- 跟进记录包含：跟进方式、沟通内容、下次跟进时间

**跟进提醒：**
- 超过3天未跟进的线索自动提醒
- 计划跟进时间到期前提醒

### 4.4 线索转化

**转化条件：**
- 客户有明确的采购意向
- 已经与决策人建立联系
- 了解客户需求和预算

**转化操作：**
1. 将线索转化为客户
2. 创建对应的商机
3. 记录转化时间和原因

**转化后：**
- 线索状态变为"已转化"
- 自动创建客户记录
- 关联联系人信息
- 创建初始商机

### 4.5 线索输单

**输单原因：**
- 客户预算不足
- 客户选择了竞品
- 客户暂时没有需求
- 无法联系到客户
- 其他原因

**输单处理：**
1. 标记线索为"已输单"
2. 记录输单原因
3. 线索退回公海池
4. 其他销售人员可以重新领取

### 4.6 线索退回公海

**退回规则：**
- 线索输单后自动退回公海
- 销售人员可以主动退回线索
- 线索超过30天未跟进自动退回公海

**公海池管理：**
- 公海池中的线索没有负责人
- 任何销售人员都可以领取
- 领取后成为该销售人员的线索

## 5. 权限控制

### 5.1 销售成员权限

- 查看自己的线索
- 创建线索
- 领取公海池线索
- 添加跟进记录
- 转化线索为客户
- 标记线索输单
- 退回线索到公海池

### 5.2 销售总监权限

- 查看所有线索
- 分配线索给销售成员
- 查看团队线索统计
- 查看公海池线索

### 5.3 系统管理员权限

- 查看所有线索
- 管理线索分配规则
- 批量导入线索
- 线索数据统计

## 6. 数据统计

### 6.1 个人统计

- 线索总数
- 新增线索数
- 转化线索数
- 转化率
- 跟进中线索数
- 输单线索数

### 6.2 团队统计

- 团队线索总数
- 团队转化率
- 各成员线索分布
- 线索来源分布
- 地区分布统计

### 6.3 时间维度统计

- 日新增线索数
- 周新增线索数
- 月新增线索数
- 转化周期分析

## 7. 业务规则

1. **线索唯一性**：同一电话号码只能创建一个线索
2. **分配限制**：销售人员最多拥有50个活跃线索
3. **跟进要求**：新增线索24小时内必须首次跟进
4. **转化时限**：线索转化周期建议不超过30天
5. **公海规则**：输单线索自动退回公海池
6. **领取限制**：销售人员每天最多领取10个公海线索

## 8. 飞书通知集成

**通知场景：**
- 线索分配通知：通知销售人员有新线索分配
- 跟进提醒：提醒销售人员及时跟进线索
- 转化成功：通知销售总监线索成功转化
- 线索输单：通知销售总监线索输单

## 9. 常见问题

**Q1: 线索转化后还能找回吗？**
A: 转化后的线索状态为"已转化"，不能恢复为跟进中状态，但可以查看转化记录。

**Q2: 如何重新分配已分配的线索？**
A: 目前只有销售总监有权限重新分配线索，需要先将线索退回公海池再重新分配。

**Q3: 公海池的线索有优先级吗？**
A: 系统默认按线索创建时间排序，最新创建的线索排在前面。

**Q4: 如何批量导入线索？**
A: 可以使用Excel模板批量导入线索，需要先下载模板，填写信息后上传。

**Q5: 线索输单后可以恢复吗？**
A: 输单后的线索退回公海池，可以重新领取，但历史记录会保留。
