# 客户管理流程

## 1. 客户生命周期

```
潜在客户 → 跟进中 → 已成交 → 跟进中
              ↓
           已输单 → 已失效（公海池）
```

## 2. 客户状态说明

| 状态 | 值 | 说明 |
|------|-----|------|
| 跟进中 | FOLLOWING | 正在跟进的客户 |
| 已成交 | WON | 已成交的客户 |
| 已输单 | LOST | 已输单的客户 |
| 已失效 | INACTIVE | 退回公海池的客户 |

## 3. 客户来源

客户主要来源于线索转化：

- 线索转化：从线索转化而来
- 客户推荐：现有客户推荐
- 市场活动：通过市场活动获取
- 其他渠道：合作伙伴、展会等

## 4. 核心功能

### 4.1 客户创建

**创建方式：**
- **线索转化**：从线索转化自动创建客户
- **手动创建**：销售人员手动录入客户信息
- **批量导入**：通过Excel批量导入客户

**必填字段：**
- 客户公司名称
- 所在城市
- 所属行业

**可选字段：**
- 公司地址
- 公司规模
- 客户来源线索ID
- 负责人

### 4.2 客户分配

**分配规则：**
1. **线索转化分配**：从线索转化时，客户归属原线索负责人
2. **手动分配**：销售总监可以手动分配客户给销售人员
3. **客户领取**：销售人员可以领取公海池中的客户

**分配权限：**
- 销售总监：可以分配任何客户给团队成员
- 销售成员：只能领取公海池客户，不能主动分配

**分配限制：**
- 每个销售人员最多拥有100个活跃客户
- 公海池中的客户可以被任何销售人员领取

### 4.3 客户信息管理

**基本信息：**
- 客户公司名称
- 所属行业
- 所在城市
- 公司地址
- 公司规模

**关联信息：**
- 客户联系人（支持多个）
- 客户商机（支持多个）
- 客户合同（支持多个）
- 客户跟进记录

**负责人信息：**
- 负责人：当前跟进的销售人员
- 创建人：最初创建客户的用户

### 4.4 客户跟进

**跟进要求：**
- 新客户3天内必须首次跟进
- 活跃客户至少每7天跟进一次
- 潜力客户至少每15天跟进一次

**跟进记录：**
- 跟进方式（电话、邮件、拜访、微信）
- 沟通内容摘要
- 客户反馈
- 下次跟进计划

**跟进提醒：**
- 超期未跟进自动提醒
- 计划跟进时间提醒

### 4.5 客户状态变更

**状态变更场景：**

1. **跟进中 → 已成交**
   - 条件：客户首次签约
   - 操作：系统自动更新或手动标记
   - 结果：客户状态变为"已成交"

2. **跟进中 → 已输单**
   - 条件：客户选择竞品或明确表示不会采购
   - 操作：标记输单并填写原因
   - 结果：客户退回公海池

3. **已成交 → 跟进中**
   - 条件：已成交客户产生新的商机
   - 操作：创建新商机
   - 结果：客户状态保持活跃

4. **跟进中 → 已失效**
   - 条件：超过30天未跟进或明确表示暂时无需求
   - 操作：销售总监确认后退回公海
   - 结果：客户退回公海池

### 4.6 客户退回公海

**退回规则：**
1. **自动退回**：客户输单后自动退回公海
2. **手动退回**：销售总监可以将客户退回公海
3. **超期退回**：超过60天未跟进的客户自动退回公海

**退回影响：**
- 客户负责人清空
- 客户状态变为"已失效"
- 客户进入公海池，其他销售人员可以领取

### 4.7 客户领取

**领取权限：**
- 销售成员可以领取公海池客户
- 每天最多领取5个客户
- 领取后成为该客户的负责人

**领取限制：**
- 领取后7天内不能退回（除非客户明确拒绝）
- 领取后必须在3天内首次跟进

## 5. 客户联系人管理

**联系人信息：**
- 联系人姓名
- 联系电话
- 邮箱
- 职位
- 部门
- 决策角色（决策者、影响者、执行者）

**联系人分类：**
- 主要联系人：主要的决策人
- 技术联系人：技术部门对接人
- 财务联系人：财务部门对接人
- 其他联系人：其他相关人员

## 6. 权限控制

### 6.1 销售成员权限

- 查看自己的客户
- 领取公海池客户
- 编辑客户基本信息
- 添加跟进记录
- 管理客户联系人
- 创建客户商机
- 查看客户合同

### 6.2 销售总监权限

- 查看团队所有客户
- 分配客户给团队成员
- 将客户退回公海
- 查看客户统计报表
- 查看团队业绩

### 6.3 系统管理员权限

- 查看所有客户
- 批量导入客户
- 管理客户分类
- 客户数据统计

## 7. 数据统计

### 7.1 个人统计

- 客户总数
- 新增客户数
- 成交客户数
- 输单客户数
- 客户活跃度
- 客户跟进率

### 7.2 团队统计

- 团队客户总数
- 团队成交客户数
- 各成员客户分布
- 客户行业分布
- 客户地区分布
- 客户规模分布

### 7.3 客户价值分析

- 高价值客户（年采购额 > 50万）
- 中等价值客户（年采购额 10-50万）
- 低价值客户（年采购额 < 10万）
- 潜力客户（未成交但有明确需求）

## 8. 业务规则

1. **客户唯一性**：同一公司名称只能创建一个客户
2. **领养限制**：销售人员每天最多领取5个公海客户
3. **容量限制**：销售人员最多拥有100个活跃客户
4. **跟进要求**：新客户3天内必须首次跟进
5. **退回规则**：输单客户自动退回公海
6. **公海保护**：领取后7天内不能无故退回

## 9. 飞书通知集成

**通知场景：**
- 客户分配通知：通知销售人员有新客户分配
- 跟进提醒：提醒销售人员及时跟进客户
- 成交通知：通知销售总监客户成功成交
- 输单通知：通知销售总监客户输单
- 退回通知：通知销售人员客户被退回公海

## 10. 常见问题

**Q1: 客户输单后可以恢复吗？**
A: 输单后的客户退回公海池，可以重新领取，但历史记录会保留。

**Q2: 如何转移客户给其他销售人员？**
A: 目前只能通过销售总监重新分配客户，或先将客户退回公海再由其他人领取。

**Q3: 客户可以重新分配吗？**
A: 可以，销售总监有权将客户重新分配给团队成员，需要填写分配原因。

**Q4: 公海池的客户如何排序？**
A: 默认按客户退回公海的时间排序，最近退回的客户排在前面。

**Q5: 如何批量导入客户？**
A: 可以使用Excel模板批量导入客户，需要先下载模板，填写信息后上传。
