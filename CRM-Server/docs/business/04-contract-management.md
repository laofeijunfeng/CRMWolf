# 合同管理流程

## 1. 合同生命周期

```
创建合同 → 草稿 → 审批中 → 待签署 → 已签署 → 已完成
                    ↓
                  已拒绝 → 草稿
                    ↓
                  已撤回 → 草稿
```

## 2. 合同状态说明

| 状态 | 值 | 说明 | 可执行操作 |
|------|-----|------|----------|
| 草稿 | DRAFT | 合同草稿，尚未提交审批 | 提交审批、编辑、删除 |
| 审批中 | PENDING_REVIEW | 合同正在审批流程中 | 查看详情、撤回 |
| 待签署 | PENDING_SIGNATURE | 审批通过，待签署 | 上传签署文件 |
| 已签署 | SIGNED | 合同已签署完成 | 查看详情 |
| 已完成 | COMPLETED | 合同已履行完毕 | 查看详情 |
| 已取消 | CANCELLED | 合同已取消 | 查看详情 |

## 3. 核心功能

### 3.1 合同创建

**创建方式：**
- **从赢单商机创建**：推荐方式，自动带出商机信息
- **手动创建**：直接创建新合同

**从赢单商机创建（推荐）：**
```bash
POST /api/v1/contracts/from-opportunity/{opportunity_id}
参数：
- contract_name: 合同名称
- signing_contact_id: 签约联系人ID
```

**自动填充信息：**
- 客户名称：从商机关联客户获取
- 合同金额：从商机实际成交金额获取
- 签约日期：使用当前日期
- 负责人：从商机负责人获取

**必填字段：**
- 合同名称
- 关联客户
- 合同金额
- 授权模式
- 签约联系人
- 签约日期

### 3.2 合同草稿

**草稿状态：**
- 合同创建后默认为"草稿"状态
- 草稿可以编辑和删除
- 草稿可以提交审批

**编辑权限：**
- 合同创建人可以编辑
- 系统管理员可以编辑

**草稿有效期：**
- 草稿建议在30天内完成并提交审批
- 超过90天的草稿系统自动提醒

### 3.3 合同提交审批

**提交条件：**
- 合同状态为"草稿"
- 合同基本信息完整
- 合同金额大于0

**审批流程：**
- 系统根据合同金额自动匹配审批流程
- 小额合同（< 10万）：销售总监审批
- 中等金额（10-50万）：销售总监 + 财务审批
- 大额合同（≥ 50万）：销售总监 + 财务 + 系统管理员审批

**提交后：**
- 合同状态变为"审批中"
- 发送通知给第一级审批人
- 合同信息锁定，不可编辑

### 3.4 合同审批

**审批操作：**

1. **审批通过**
   - 如果是最终节点：合同状态变为"待签署"
   - 如果有下一节点：通知下一级审批人
   - 通知提交人审批进度

2. **审批拒绝**
   - 合同状态退回"草稿"
   - 记录拒绝原因
   - 通知提交人被拒绝

**审批权限：**
- 基于角色的权限控制
- 当前节点的审批角色才能审批
- 预置角色：SALES_DIRECTOR、SYSTEM_ADMIN

**审批记录：**
- 所有审批操作都会记录
- 包含审批人、审批时间、审批意见
- 支持查询完整审批历史

### 3.5 合同撤回

**撤回条件：**
- 合同状态为"审批中"
- 只有提交人可以撤回

**撤回后：**
- 合同状态退回"草稿"
- 审批流程终止
- 可以修改后重新提交

### 3.6 合同签署

**待签署状态：**
- 审批全部通过后进入此状态
- 需要上传已签署的合同文件

**签署操作：**
1. 上传已签署的合同扫描件
2. 填写实际签署日期
3. 合同状态变为"已签署"

**签署要求：**
- 必须上传合同文件
- 文件格式：PDF、JPG、PNG
- 文件大小限制：10MB以内

### 3.7 合同履行

**已签署状态：**
- 合同已签署完成
- 进入履行阶段
- 可以添加履行记录

**履行跟踪：**
- 发货记录
- 付款记录
- 验收记录
- 售后服务

**合同完成：**
- 所有条款履行完毕
- 合同状态变为"已完成"

### 3.8 合同取消

**取消条件：**
- 审批中的合同可以撤回
- 未签署的合同可以取消
- 已签署的合同需要特殊流程

**取消操作：**
- 系统管理员可以取消合同
- 需要填写取消原因
- 通知相关人员

## 4. 合同信息管理

**负责人信息：**

- 创建时间

### 5.2 销售总监权限

- 审批合同
- 查看合同统计

### 5.3 系统管理员权限

- 查看所有合同
- 审批合同
- 管理审批流程
- 取消合同
- 合同数据统计

## 6. 审批流程详解

### 6.1 预置审批流程

**小额合同审批（< 10万）**
```
提交 → 销售总监审批 → 通过（待签署）/ 拒绝（草稿）
```

**中等金额合同（10-50万）**
```
提交 → 销售总监审批 → 财务审批 → 通过（待签署）/ 拒绝（草稿）
```

**大额合同（≥ 50万）**
```
提交 → 销售总监审批 → 财务审批 → 系统管理员审批 → 通过（待签署）/ 拒绝（草稿）
```

### 6.2 审批权限

**销售总监审批权限：**
- 审批所有金额的合同
- 关注合同金额合理性
- 关注客户资质
- 关注合同条款

**财务审批权限：**
- 审核付款条款
- 审核价格政策
- 审核发票信息
- 审核风险条款

**系统管理员审批权限：**
- 大额合同最终审批
- 审核合同合规性
- 审核特殊条款
- 批准特殊授权

### 6.3 审批规则

1. **顺序审批**：按节点顺序依次审批
2. **全员通过**：所有节点都必须通过才算通过
3. **一票否决**：任一节点拒绝则整体拒绝
4. **超时提醒**：审批超期自动提醒

## 8. 业务规则

1. **创建方式**：优先从赢单商机创建合同
2. **金额限制**：合同金额必须大于0
3. **审批必选**：所有合同必须经过审批流程
4. **签署必填**：审批通过后必须上传签署文件
5. **状态锁定**：审批中的合同不可编辑
6. **撤回限制**：只有提交人可以撤回审批
7. **取消权限**：只有系统管理员可以取消合同
8. **编号唯一**：合同编号自动生成，全局唯一

## 9. 飞书通知集成

**通知场景：**
- 提交审批通知：通知第一级审批人
- 审批通过通知：通知下一级审批人或提交人
- 审批拒绝通知：通知提交人
- 合同签署通知：通知相关人员
- 审批超期提醒：提醒审批人及时审批

## 10. 常见问题

**Q1: 合同创建后可以修改吗？**
A: 草稿状态可以修改，审批中的合同不能修改，需要先撤回。

**Q2: 审批拒绝后怎么办？**
A: 审批拒绝后合同退回草稿，可以修改后重新提交审批。

**Q3: 如何加快审批流程？**
A: 可以提醒审批人及时处理，或者调整审批流程配置。

**Q4: 合同编号可以自定义吗？**
A: 合同编号由系统自动生成，不支持自定义，但可以查询。

**Q5: 可以创建没有商机的合同吗？**
A: 可以，但推荐从赢单商机创建，这样可以保证数据一致性。

**Q6: 合同签署后可以修改吗？**
A: 已签署的合同不能直接修改，如需变更需要创建补充协议。
