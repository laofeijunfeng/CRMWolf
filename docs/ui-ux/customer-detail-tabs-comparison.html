客户详情页标签页改进方案对比
💡 方案A：分组折叠
✓ 核心标签宽度≥100px，触摸目标充足
✓ 高频操作（跟进、联系人）直达
✓ 下拉菜单收纳低频功能，认知负荷低
✓ 移动端下拉改为底部弹层，适配友好
⚠ 实现成本：低（仅需调整标签结构）
💡 方案B：底部固定操作栏
✓ 操作前置，无需切换标签页
✓ 移动端体验极佳（底部符合拇指热区）
✓ 信息密度高，一眼看到关键信息
⚠ 桌面端信息密度过高，需优化布局
⚠ 实现成本：高（需重构页面布局）
💡 方案C：左侧导航栏
✓ 垂直导航，触摸目标48×48dp
✓ 桌面端信息密度最佳
✓ 导航始终可见，无需回到顶部
⚠ 移动端需改为抽屉/下拉菜单
⚠ 实现成本：中（需新增导航组件）
